# PokeApp

PokeApp es una aplicación creada con Flutter, busca conectarte al mundo pokemon a través de su poderoso uso de Api y GoogleFireBase.

## Información

Esta aplicación utiliza los siguientes paquetes:

- firebase_core
- firebase_auth
- rflutter_alert
- pokeapi
